OndoMathX

OndoMathX is a C++ research finite element solver developed primarily by myself, with contributions from several PhD students. It implements the Spectral Element Method (SEM) on hexahedral meshes for the efficient simulation of time-dependent wave propagation problems.

Key features

High-order spectral elements: tensor-product Gauss-Lobatto quadrature on hexahedral meshes, exact mass-lumping yielding diagonal mass matrices, high-order geometric accuracy via parametric meshes.

Non-assembled computations: matrix-free strategy exploiting the tensor-product structure for efficient matrix-vector products, low memory footprint.

Implicit-explicit (IMEX) time schemes: time integration adapted to multi-scale hyperbolic problems, with provable stability properties independent of the stiffness parameter.

SEM-FFT coupling: high-order spectral element FFT for efficient solving of elliptic problems on structured grids [22].

Perfectly Matched Layers (PML): absorbing boundary conditions for wave propagation in unbounded domains, including anisotropic and stabilized variants [1, 9, 23].

Boundary stabilization: high-frequency parasitic mode suppression for exponentially stable observer discretizations [39].

Applications implemented

• Isotropic and incompressible elastodynamics

• Nearly-incompressible hyperelastic solids (cardiac tissue, cornea) [44, 45]

• Acoustoelasticity: wave propagation in pre-stressed media [37]

• Electromagnetism in coaxial cables [33]

• Acoustic and gravito-acoustic wave propagation [35, 40]

• Cardiac shear-wave elastography: ARF modelling [26]

• Ultrasonic NDT in composite plates [34] and deformed pipes [37]

Industrial transfer

OndoMathX is developed in close scientific collaboration with CEA-List/DISC (Alexandre Imperiale), which integrates a parallel SEM kernel into the commercial inspection simulation software CIVA. Two CEA-funded industrial studies on wave diffraction from rough surfaces (~20k€ each) have directly leveraged the methods developed in OndoMathX.
