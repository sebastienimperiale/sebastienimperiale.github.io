# Sébastien Imperiale — Personal Website

- Source code for my personal academic website, built with Jekyll, HTML, CSS, and JavaScript.
- Includes pages on research, publications, students, talks, teaching, and software.

## Structure

- `index.html` — Homepage
- `publications.html`, `research.html`, `students.html`, `talks.html`, `teaching.html`, `software.html` — Individual pages
- `assets/` — All static files:
  - `css/` — Stylesheets
  - `images/` — General images
  - `js/` — JavaScript
  - `webfonts/` — Custom fonts

## Notes

- Static site generated with Jekyll and hosted on GitHub Pages.
- To update, edit the HTML/Markdown files and push to the `main` branch.
